# Contributing to AI Skill File Generator

First off, thank you for considering contributing to the AI Skill File Generator! It's people like you that make the open-source community such a great place to learn, inspire, and create.

## Code of Conduct

By participating in this project, you are expected to uphold our Code of Conduct. Please report unacceptable behavior to the project maintainers.

## How Can I Contribute?

### Reporting Bugs

- **Check if the bug has already been reported** by searching on GitHub under Issues.
- If you can't find an open issue that describes the problem, **open a new one**. Include a title and clear description, as much relevant information as possible, and a code sample or an executable test case demonstrating the expected behavior that is not occurring.

### Suggesting Enhancements

Enhancement suggestions are tracked as GitHub issues. When creating an enhancement suggestion, please:

- **Use a clear and descriptive title** for the issue to identify the suggestion.
- **Provide a step-by-step description of the suggested enhancement** in as many details as possible.
- **Describe the current behavior and explain which behavior you expected to see instead** and why.

### Pull Requests

- Fill in the required template
- Do not include issue numbers in the PR title
- Include screenshots and animated GIFs in your pull request whenever possible.
- Follow the existing code style and conventions.
- Document any new features or changes to existing functionality.

## Style Guide

- We use standard JavaScript/React conventions.
- Use meaningful variable and function names.
- Keep components small and focused.
- Ensure all new features are documented in the README.
